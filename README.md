📌 Internship Overview

This repository contains my completed tasks for the **Cyber Security Internship Program** under Future Interns.

The objective of this internship is to gain hands-on experience in:
- Vulnerability Assessment
- Phishing Detection
- API Security Analysis
- Security Reporting & Risk Classification

---

🧪 Tasks Completed

🔹 Task 1: Vulnerability Assessment
- Target: `scanme.nmap.org`
- Tools Used: Nmap, OWASP ZAP
- Key Activities:
  - Network scanning & service enumeration
  - Web vulnerability scanning
  - Risk classification using OWASP Top 10
- Key Findings:
  - Outdated Apache & OpenSSH
  - Missing security headers
  - No HTTPS (MITM risk)
  - Directory browsing enabled

📄 Report: `VA_Report_Hariraj_S.docx`

---

🔹 Task 2: Phishing Detection & Analysis
- Target: `login.microstfonline.com`
- Tools Used: URLScan.io, VirusTotal
- Key Activities:
  - Phishing URL investigation
  - Typosquatting detection
  - Redirect chain analysis
  - Threat intelligence correlation
- Key Findings:
  - Microsoft impersonation attack
  - Redirect abuse technique
  - Suspicious infrastructure (DigitalOcean IP)
  - Confirmed malicious (Sophos detection)

📄 Report: `Phishing_Report_Hariraj_S.docx`

---

🔹 Task 3: API Security Risk Analysis
- APIs Tested:
  - jsonplaceholder.typicode.com
  - reqres.in
- Tools Used: Postman, cURL, Browser DevTools
- Framework: OWASP API Security Top 10 (2023)

- Key Findings:
  - No authentication on sensitive endpoints
  - IDOR vulnerability
  - Sensitive data exposure
  - Missing rate limiting
  - Input validation issues

📄 Report: `API_Security_Risk_Analysis_Report_Final.docx`
